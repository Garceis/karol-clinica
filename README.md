# Site de Apresentação | Dra. Karol Silva - Neuropsicologia

Este é um site institucional fictício para apresentação de uma profissional da área de neuropsicologia. O site é responsivo, moderno e utiliza apenas HTML, CSS e JavaScript puro.

## Estrutura
- **index.html**: Página principal com seções de Início, Sobre, Serviços, Depoimentos e Contato.
- **style.css**: Estilos modernos e responsivos.
- **script.js**: Script para envio do formulário de contato.
- **assets/**: Pasta para imagens ilustrativas (substitua pelos arquivos reais desejados).

## Como usar
1. Substitua as imagens da pasta `assets` por imagens reais de sua preferência.
2. Edite os textos conforme necessário.
3. Abra o arquivo `index.html` em seu navegador.

## Observações
- Este projeto não utiliza frameworks ou bibliotecas externas.
- Todos os textos e imagens são fictícios e podem ser personalizados.
